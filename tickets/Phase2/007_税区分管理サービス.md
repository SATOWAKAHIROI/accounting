# チケット #007: 税区分管理サービス

## ステータス
- [x] 完了

## 機能概要
税区分のCRUD操作、有効期間別検索、税率履歴管理を提供するサービスクラス。

## 実装ファイル
- [x] `service/TaxTypeService.java`
- [x] `dto/request/TaxTypeRequest.java`（新規作成）
- [x] `dto/response/TaxTypeResponse.java`（新規作成）

## 主要メソッド
- `List<TaxTypeResponse> findAll(Long companyId)` - 全件取得
- `TaxTypeResponse findById(Long id)` - ID検索
- `TaxTypeResponse create(Long companyId, TaxTypeRequest request)` - 作成
- `TaxTypeResponse update(Long id, TaxTypeRequest request)` - 更新
- `void delete(Long id)` - 削除
- `List<TaxTypeResponse> findEffectiveTaxTypes(Long companyId, LocalDate date)` - 有効税区分取得

## ビジネスルール
- 同一会社内でコードが重複しないこと
- 有効期間（effectiveFrom、effectiveTo）の妥当性チェック
- 削除時、仕訳明細・請求書明細で使用されているかチェック
- 税率は0以上100以下

## 依存関係
- TaxTypeRepository
- JournalDetailRepository
- InvoiceDetailRepository
