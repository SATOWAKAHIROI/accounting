## 機能概要
ユーザーのREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/UserController.java`
- [x] `controller/AuthController.java`（認証用）

## エンドポイント

### UserController
- `GET /api/users` - 一覧取得
- `GET /api/users/{id}` - 詳細取得
- `POST /api/users` - 作成
- `PUT /api/users/{id}` - 更新
- `DELETE /api/users/{id}` - 削除

### AuthController
- `POST /api/auth/login` - ログイン
  - リクエストボディ: `{ "email": "...", "password": "..." }`
  - レスポンス: `{ "token": "...", "user": {...} }`
- `POST /api/auth/logout` - ログアウト
- `POST /api/auth/refresh` - トークンリフレッシュ（オプション）

## セキュリティ
- ログインエンドポイント以外は認証必須
- JWTトークンを検証するフィルター実装
