## 機能概要
仕訳のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/JournalController.java`

## エンドポイント
- `GET /api/companies/{companyId}/journals` - 一覧取得
- `GET /api/companies/{companyId}/journals/{id}` - 詳細取得
- `POST /api/companies/{companyId}/journals` - 作成
- `PUT /api/companies/{companyId}/journals/{id}` - 更新
- `DELETE /api/companies/{companyId}/journals/{id}` - 削除
- `GET /api/companies/{companyId}/journals?startDate={start}&endDate={end}` - 日付範囲検索
