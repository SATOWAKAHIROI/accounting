# チケット #017: 会計期間管理サービス

## ステータス
- [x] 完了

## 機能概要
会計期間のCRUD操作、期間締め/再開を提供するサービスクラス。

## 実装ファイル
- [x] `service/FiscalPeriodService.java`
- [x] `dto/request/FiscalPeriodRequest.java`（新規作成）
- [x] `dto/response/FiscalPeriodResponse.java`（新規作成）

## 主要メソッド
- `List<FiscalPeriodResponse> findAll(Long companyId)` - 全件取得
- `FiscalPeriodResponse findById(Long id)` - ID検索
- `FiscalPeriodResponse create(Long companyId, FiscalPeriodRequest request)` - 作成
- `FiscalPeriodResponse update(Long id, FiscalPeriodRequest request)` - 更新
- `void closePeriod(Long id)` - 期間締め
- `void reopenPeriod(Long id)` - 期間再開

## ビジネスルール
- 期間の重複チェック（同一会社内で日付範囲が重複しない）
- 開始日 < 終了日の検証
- 期間締め時、その期間の仕訳が存在することを確認
- 締められた期間の仕訳は編集・削除不可
- 期間再開は慎重に実施（監査上の理由で制限を検討）

## 依存関係
- FiscalPeriodRepository
- JournalRepository
