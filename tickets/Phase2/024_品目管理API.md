## 機能概要
品目のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/ItemController.java`

## エンドポイント
- `GET /api/companies/{companyId}/items` - 一覧取得
- `GET /api/companies/{companyId}/items/{id}` - 詳細取得
- `POST /api/companies/{companyId}/items` - 作成
- `PUT /api/companies/{companyId}/items/{id}` - 更新
- `DELETE /api/companies/{companyId}/items/{id}` - 削除
