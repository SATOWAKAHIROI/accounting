## ステータス
- [x] 完了

## 機能概要
税区分のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/TaxTypeController.java`

## エンドポイント
- `GET /api/companies/{companyId}/tax-types` - 一覧取得
- `GET /api/companies/{companyId}/tax-types/{id}` - 詳細取得
- `POST /api/companies/{companyId}/tax-types` - 作成
- `PUT /api/companies/{companyId}/tax-types/{id}` - 更新
- `DELETE /api/companies/{companyId}/tax-types/{id}` - 削除
- `GET /api/companies/{companyId}/tax-types/effective?date={date}` - 有効税区分取得
