# チケット #028: レポート生成API

## ステータス
- [x] 完了

## 機能概要
各種帳票のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/ReportController.java`

## エンドポイント
- `GET /api/companies/{companyId}/reports/general-ledger` - 総勘定元帳
  - パラメータ: `accountId`, `startDate`, `endDate`, `format`（pdf/excel）
- `GET /api/companies/{companyId}/reports/trial-balance` - 試算表
  - パラメータ: `asOfDate`, `format`（pdf/excel）
- `GET /api/companies/{companyId}/reports/profit-loss` - 損益計算書
  - パラメータ: `startDate`, `endDate`, `format`（pdf/excel）
- `GET /api/companies/{companyId}/reports/balance-sheet` - 貸借対照表
  - パラメータ: `asOfDate`, `format`（pdf/excel）

## レスポンス
- JSON形式: レポートデータを返却
- PDF/Excel形式: バイナリデータを返却（Content-Type: application/pdf or application/vnd.ms-excel）

## 依存関係
- GeneralLedgerService
- TrialBalanceService
- ProfitLossService
- BalanceSheetService
