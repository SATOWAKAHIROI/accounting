# チケット #018: 会社管理サービス

## ステータス
- [x] 完了

## 機能概要
会社情報のCRUD操作を提供するサービスクラス。

## 実装ファイル
- [x] `service/CompanyService.java`
- [x] `dto/request/CompanyRequest.java`（新規作成）
- [x] `dto/response/CompanyResponse.java`（新規作成）

## 主要メソッド
- `List<CompanyResponse> findAll()` - 全件取得
- `CompanyResponse findById(Long id)` - ID検索
- `CompanyResponse create(CompanyRequest request)` - 作成
- `CompanyResponse update(Long id, CompanyRequest request)` - 更新
- `void delete(Long id)` - 削除

## ビジネスルール
- 会社名は必須
- 削除時、関連する会計データが存在する場合は削除不可
- 削除時、ユーザーとの紐付け（UserCompanyRole）も削除

## 依存関係
- CompanyRepository
- UserCompanyRoleRepository
- AccountRepository（削除時の参照チェック用）
