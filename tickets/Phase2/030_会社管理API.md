## 機能概要
会社情報のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/CompanyController.java`

## エンドポイント
- `GET /api/companies` - 一覧取得
- `GET /api/companies/{id}` - 詳細取得
- `POST /api/companies` - 作成
- `PUT /api/companies/{id}` - 更新
- `DELETE /api/companies/{id}` - 削除
