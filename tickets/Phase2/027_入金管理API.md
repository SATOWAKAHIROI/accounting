## 機能概要
入金のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/PaymentController.java`

## エンドポイント
- `GET /api/companies/{companyId}/payments` - 一覧取得
- `GET /api/companies/{companyId}/payments/{id}` - 詳細取得
- `POST /api/companies/{companyId}/payments` - 作成
- `PUT /api/companies/{companyId}/payments/{id}` - 更新
- `DELETE /api/companies/{companyId}/payments/{id}` - 削除
