## 機能概要
請求書のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/InvoiceController.java`

## エンドポイント
- `GET /api/companies/{companyId}/invoices` - 一覧取得
- `GET /api/companies/{companyId}/invoices/{id}` - 詳細取得
- `POST /api/companies/{companyId}/invoices` - 作成
- `PUT /api/companies/{companyId}/invoices/{id}` - 更新
- `DELETE /api/companies/{companyId}/invoices/{id}` - 削除
- `PATCH /api/companies/{companyId}/invoices/{id}/status` - ステータス更新
- `GET /api/companies/{companyId}/invoices/{id}/pdf` - PDF生成
