## ステータス

## 機能概要
会計期間のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/FiscalPeriodController.java`

## エンドポイント
- `GET /api/companies/{companyId}/fiscal-periods` - 一覧取得
- `GET /api/companies/{companyId}/fiscal-periods/{id}` - 詳細取得
- `POST /api/companies/{companyId}/fiscal-periods` - 作成
- `PUT /api/companies/{companyId}/fiscal-periods/{id}` - 更新
- `PATCH /api/companies/{companyId}/fiscal-periods/{id}/close` - 期間締め
- `PATCH /api/companies/{companyId}/fiscal-periods/{id}/reopen` - 期間再開
