# チケット #008: 取引先管理サービス

## ステータス
- [x] 完了

## 機能概要
取引先のCRUD操作、タイプ別検索（顧客/仕入先）を提供するサービスクラス。

## 実装ファイル
- [x] `service/PartnerService.java`
- [x] `dto/request/PartnerRequest.java`（新規作成）
- [x] `dto/response/PartnerResponse.java`（新規作成）

## 主要メソッド
- `List<PartnerResponse> findAll(Long companyId)` - 全件取得
- `PartnerResponse findById(Long id)` - ID検索
- `PartnerResponse create(Long companyId, PartnerRequest request)` - 作成
- `PartnerResponse update(Long id, PartnerRequest request)` - 更新
- `void delete(Long id)` - 削除
- `List<PartnerResponse> findByPartnerType(Long companyId, PartnerType type)` - タイプ別検索

## ビジネスルール
- 同一会社内でコードが重複しないこと
- 取引先タイプ: CUSTOMER（顧客）、VENDOR（仕入先）、BOTH（両方）
- 削除時、仕訳明細・請求書・入金で使用されているかチェック
