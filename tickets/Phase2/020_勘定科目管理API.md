# チケット #020: 勘定科目管理API

## ステータス
- [x] 完了

## 機能概要
勘定科目のREST APIエンドポイントを提供するコントローラークラス。

## 実装ファイル
- [x] `controller/AccountController.java`

## エンドポイント

### `GET /api/companies/{companyId}/accounts`
- 一覧取得
- クエリパラメータ: `accountType`（オプション）
- レスポンス: `ApiResponse<List<AccountResponse>>`

### `GET /api/companies/{companyId}/accounts/{id}`
- 詳細取得
- レスポンス: `ApiResponse<AccountResponse>`

### `POST /api/companies/{companyId}/accounts`
- 作成
- リクエストボディ: `AccountRequest`
- レスポンス: `ApiResponse<AccountResponse>`
- HTTPステータス: 201 Created

### `PUT /api/companies/{companyId}/accounts/{id}`
- 更新
- リクエストボディ: `AccountRequest`
- レスポンス: `ApiResponse<AccountResponse>`

### `DELETE /api/companies/{companyId}/accounts/{id}`
- 削除
- レスポンス: `ApiResponse<Void>`
- HTTPステータス: 204 No Content

## 実装のポイント
- `@RestController` + `@RequestMapping("/api/companies/{companyId}/accounts")`
- `@Valid`でリクエストボディのバリデーション
- AccountServiceを注入して処理を委譲
- 例外はGlobalExceptionHandlerで処理

## 依存関係
- AccountService
