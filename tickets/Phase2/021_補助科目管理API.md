## 機能概要
補助科目のREST APIエンドポイントを提供。

## 実装ファイル
- [x] `controller/SubAccountController.java`

## エンドポイント
- `GET /api/companies/{companyId}/sub-accounts` - 一覧取得
- `GET /api/companies/{companyId}/sub-accounts/{id}` - 詳細取得
- `POST /api/companies/{companyId}/sub-accounts` - 作成
- `PUT /api/companies/{companyId}/sub-accounts/{id}` - 更新
- `DELETE /api/companies/{companyId}/sub-accounts/{id}` - 削除
